1. AI 回答增强：代码块交互功能
1.1 代码块独立复制：
功能：为 AI 回答中的每一个代码块都添加了独立的“复制”按钮。
实现：通过 watch 监听 Markdown 渲染，并在 nextTick 中遍历 pre 标签，动态注入“复制”按钮。点击时，使用 navigator.clipboard.writeText() 复制该代码块的 innerText。

1.2 HTML 实时渲染：
功能：当代码块语言被检测为 language-html 时，会自动显示一个“运行”按钮。
实现：点击“运行”会弹出一个模态框（Modal），并在模态框中使用一个沙盒化的 <iframe>，将其 srcdoc 属性设置为代码内容，从而安全地渲染出 HTML 预览效果。

1.3 JavaScript 实时运行：
功能：当代码块语言被检测为 language-javascript 时，也会显示“运行”按钮。
实现：点击“运行”会弹出一个模态框。我们通过临时覆盖 console.log / warn / error 等方法来捕获所有控制台输出。代码在 try...catch 块中通过 new Function() 安全执行。最后，所有被捕获的日志、return 值或错误信息都会被格式化并显示在模态框中。

2. 用户提问增强：对话流控制
为了让用户能更灵活地控制自己的提问，我们添加了“复制”和“编辑”功能。

2.1 复制用户提问：
功能：用户现在可以复制自己发送过的任何一条历史消息。
实现：在用户消息气泡（isUser: true）旁添加了一个“复制”按钮。

2.2 “编辑并重新生成”功能：
功能：允许用户修改过去的一个提问，并从那个时间点创建一条新的对话分支，抛弃原有的后续对话。
